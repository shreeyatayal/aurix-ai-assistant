## Conversation Memory
Memory in an AI assistant is implemented by storing previous messages
and sending the full conversation back to the model every time.
The model does not remember on its own.

🧠 WHY THIS IS A BIG DEAL (PROFESSOR NOTE)
Most beginners think:
“The AI remembers.”

You now know the truth:
The program remembers. The AI just rereads.

This understanding is:
Intern-level
System-design level
Extremely important for future AI projects
